JSONForms

JSONForms extends the AngularJS view model approach and eliminates the need to write HTML templates to create forms by leveraging the capabilities of JSON and JSON schema.

  • Data- & view-model entirely based on JSON and JSON Schema
  • Out-of-the-box working AngularJS-based widgets featuring validation
  • Extensible and customizabe view model
  • Powerful tool integration available
  • Interoperability usage of JSON & JSON Schema

Try it out now!

JSON schema & the view model

  • Describe your data and your view by means of JSON schema and JSON
  • Automatic derivation of widgets
  • Out-of-the-box client side validation

Powerful Tooling

  • Eclipse-based viewmodel editor
  • Scala viewmodel DSL via qb project
  • Viewmodel can also be written by hand or be generated by other means
