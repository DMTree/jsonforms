This example shows the available renderers for different layouts.

VerticalLayout:
This layouts all elements under each other.

HorizontalLayout:
This layouts all elements next to each other.

Group:
A group behaves like a VerticalLayout with a custom heading.

Complex example:
This example consists of a Group (green) which contains a HorizontalLayout (red), which contains two VerticalLayout (blue). Each of the VerticalLayouts contains two controls.
