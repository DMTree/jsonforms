The uischema, schema and data attributes of the jsonforms expect values that represent the UI schema, the JSON schema and the actual data to be rendered, respectively. But the values do not necessarily need to present when rendering is triggered. It is also possible to pass promises or functions returning promises to the directive.

In the following this is demonstrated: Click 'Load Data' and 'Load Schema' to load the JSON schema and the UI schema asynchronously. Then click 'Load UI' to finally load the rendered form.
