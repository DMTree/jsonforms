JSONForms eliminates the need to write HTML templates by hand to create customizable forms by leveraging the capabilities of JSON and JSON schema.

  • Data & UI schema entirely based on JSON and JSON Schema
  • Out-of-the-box working AngularJS-based widgets featuring validation
  • Extensible and customizable UI schema
  • Powerful tool integration available
  • Interoperability usage of JSON & JSON Schema

Try it out now!

JSON schema & the view model

  • Describe your data and your view by means of JSON schema and JSON
  • Automatic derivation of widgets
  • Out-of-the-box client side validation


Powerful Tooling

  • Eclipse-based UI schema editor
  • UI schema can also be written by hand or be generated by other means
Eclipse-based Editor


Extensible & Customizable

  • Wide range of available layouts & widgets
  • Widgets can be replaced and added easily
  • UI schema allows for technology agnostic approach

Embeddable

  • Customizable REST integration
  • Compatible with existing frameworks like Twitter Bootstrap or AngularJS
